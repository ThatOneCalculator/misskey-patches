# misskey-patches
The patches used on https://stop.voring.me

- Favicon
- Error images
- CW in replies on by default
- Change default emojis


### Usage:
Place `make.sh` into your Misskey directory, make it executable, then run `./make.sh` to patch and build Misskey.
