# misskey-patches
Very simple image/content patch script for my Misskey instance, https://stop.voring.me

__Requirements:__
- misskey
- npm, yarn
- bash
- fd (as `fdfind`)

__Usage:__
Place `make.sh` into your Misskey directory, make it executable, then run `./make.sh` to patch and build Misskey.
